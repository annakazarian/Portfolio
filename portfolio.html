Theme 1: Web Design

The first theme was about developing our first website.
Develop a responsive website through iterative processes and by implementing basic design principles,
valid code and practices for user interface design, user experiences and visual design of a website.
Learn how to work with Brackets, Cyberduck, how to use validate and debug HTML and CSS
according to current web standards using W3C Validators, inspect and text your code using Developer Tools(Google Chrome).
Overall, looking at my first repsonsive website, I did everything i understood at the beginning,
but time pass by and now i can see my mistakes.I have not used flexible grids, flexible images and media queries to make overall layout responsive.

Theme 2: Basic Animation


In these theme we mostly learned how to get hand in hand with Adobe Illustrator.
